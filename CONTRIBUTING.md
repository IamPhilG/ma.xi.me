# Contributing

mA.xI.me est un repo de méthode personnelle — il représente le workflow d'un développeur
avec Claude Code. Il est publié à titre de partage, pas comme un produit maintenu.

## Ce qui est bienvenu

- **Corrections de documentation** — typos, liens cassés, instructions ambiguës
- **Notes de compatibilité** — expérience d'installation sur macOS/Linux, changements
  de comportement suite à une mise à jour de Claude Code
- **Bug reports** sur `install.ps1` ou le hook `block-destructive-bash.sh`

## Ce qui ne sera probablement pas accepté

- Demandes de fonctionnalités — le système évolue en fonction d'un usage personnel
- PRs qui généralisent `CLAUDE.md` (Les utilisateurs qui souhaitent adapter le système
  doivent forker le repo)
- Ajout de skills ou agents sans lien avec la méthode SPEC/VERIFY/REVIEW existante

## Comment contribuer

1. Ouvrir une issue avant toute PR non triviale
2. Garder les PRs petites et ciblées (un seul sujet par PR)
3. Pour les skills ou agents : décrire le cas d'usage dans la PR

## Pas de SLA de support

Ce repo n'offre aucune garantie de réponse ou de délai de traitement.
