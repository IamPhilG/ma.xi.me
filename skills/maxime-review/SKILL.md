# mA.xI.me — Revue

À utiliser pour une revue de changement, un audit ou une analyse isolée.

1. Définir précisément le périmètre et le niveau de lecture seule attendu.
2. Examiner les fichiers, le diff et les validations disponibles.
3. Présenter les constats par sévérité avec preuves et impacts.
4. Distinguer les faits vérifiés des hypothèses.
5. Exécuter le test ou contrôle le plus pertinent avant toute action qui découle de la revue.

Ne pas modifier de fichier pendant une revue en lecture seule.
