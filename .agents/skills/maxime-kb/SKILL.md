# mA.xI.me — Knowledge base

À utiliser lorsqu'une knowledge base versionnée est disponible pour un thème pertinent.

1. Vérifier que `knowledge-base/` est disponible et initialisé.
2. Lire son index, puis seulement les fiches actives correspondant au thème.
3. Ne pas charger les archives sans demande explicite.
4. Séparer strictement le savoir générique réutilisable des données de projet, client, employeur ou secrets.
5. Proposer un enrichissement seulement si le savoir est durable, transversal et publiable.

Si la knowledge base est absente, le signaler sans inventer son contenu.
